# Operations Dashboard

This is a static dashboard that can be published with GitHub Pages.

## Files

- `index.html` contains the page structure.
- `styles.css` controls the layout and design.
- `script.js` contains the sample dashboard data and chart rendering.

## Publish on GitHub Pages

1. Create a new repository on GitHub.
2. Upload these files to the repository.
3. Open the repository settings.
4. Go to Pages.
5. Publish from the `main` branch.

After GitHub finishes publishing, the dashboard will be available at the Pages URL shown in the repository settings.
